## Installation

### Setting up Locally

1. Copy the `.env.example` to `.env.local`.

    ```bash
    cp .env.example .env.local
    ```

2. Install Packages via npm.

    ```bash
    npm install
    ```

3. Run the React JS Development Server.

    ```bash
    npm run dev
    ```
